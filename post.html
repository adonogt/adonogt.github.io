setting up development environment on arch linux

january 15, 2025

introduction

This guide shows how to set up a complete development environment on Arch Linux, covering everything from basic tool installation to IDE and text editor configuration for different programming languages.

We'll cover a setup that works for both web development and low-level programming, using tools that integrate well with the Arch ecosystem.

basic tools installation

First, let's install the essential tools that serve as the foundation for any development environment.

# Update the system
sudo pacman -Syu

# Install basic development tools
sudo pacman -S base-devel git curl wget

# Install yay for AUR access
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

configuring git

Set up your Git credentials so commits are properly identified:

# Configure name and email
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Set default editor
git config --global core.editor nvim

# Configure useful aliases
git config --global alias.st status
git config --global alias.co checkout
git config --global alias.br branch
git config --global alias.ci commit

web development environment

For web development, we need Node.js, npm, and some additional tools that streamline the workflow.

node.js and version managers

Use fnm to manage Node.js versions efficiently:

# Install fnm (Fast Node Manager)
yay -S fnm

# Add to bashrc or zshrc
echo 'eval "$(fnm env)"' >> ~/.bashrc

# Reload shell
source ~/.bashrc

# Install LTS version of Node.js
fnm install --lts
fnm use lts-latest

# Verify installation
node --version
npm --version

web development tools

Install essential tools for modern web development:

# Install alternative package managers
npm install -g yarn pnpm

# Install build and development tools
npm install -g vite webpack parcel

# Install linters and formatters
npm install -g eslint prettier

# Install testing tools
npm install -g jest vitest

neovim configuration

Configure Neovim as a complete IDE with LSP support, syntax highlighting, and essential plugins.

# Install Neovim and dependencies
sudo pacman -S neovim nodejs npm python-pip ripgrep fd

# Install Python dependencies
pip install pynvim

# Create configuration directory
mkdir -p ~/.config/nvim

basic neovim configuration

Create Neovim's main configuration file:

# ~/.config/nvim/init.lua

-- Basic configurations
vim.opt.number = true
vim.opt.relativenumber = true
vim.opt.tabstop = 2
vim.opt.shiftwidth = 2
vim.opt.expandtab = true
vim.opt.smartindent = true
vim.opt.wrap = false
vim.opt.swapfile = false
vim.opt.backup = false
vim.opt.hlsearch = false
vim.opt.incsearch = true
vim.opt.termguicolors = true
vim.opt.scrolloff = 8
vim.opt.signcolumn = "yes"
vim.opt.isfname:append("@-@")
vim.opt.updatetime = 50

-- Set leader key
vim.g.mapleader = " "

-- Basic keymaps
vim.keymap.set("n", "pv", vim.cmd.Ex)
vim.keymap.set("v", "J", ":m '>+1gv=gv")
vim.keymap.set("v", "K", ":m '<-2gv=gv")
vim.keymap.set("n", "J", "mzJ`z")
vim.keymap.set("n", "", "zz")
vim.keymap.set("n", "", "zz")

programming languages

Configure support for major programming languages using the compilers and interpreters available in Arch.

python

Set up a robust Python environment with virtual environment support:

# Python comes pre-installed on Arch, but let's install additional tools
sudo pacman -S python-virtualenv python-pip python-pipx

# Install Python development tools
pipx install black
pipx install flake8
pipx install mypy
pipx install poetry

# Create virtual environment for projects
python -m venv ~/venv/dev
source ~/venv/dev/bin/activate

# Install essential packages in virtual environment
pip install jupyter ipython requests numpy pandas

rust

Configure Rust environment for systems development:

# Install Rust via rustup
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Reload shell
source ~/.cargo/env

# Install additional components
rustup component add rust-src
rustup component add rust-analyzer
rustup component add clippy rustfmt

# Verify installation
rustc --version
cargo --version

go

Configure Go environment for application development:

# Install Go
sudo pacman -S go

# Configure GOPATH and GOBIN
echo 'export GOPATH=$HOME/go' >> ~/.bashrc
echo 'export PATH=$PATH:$GOPATH/bin' >> ~/.bashrc
source ~/.bashrc

# Install essential Go tools
go install golang.org/x/tools/gopls@latest
go install golang.org/x/tools/cmd/goimports@latest
go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest

# Verify installation
go version

containers and virtualization tools

Configure Docker and other containerization tools for modern development:

# Install Docker
sudo pacman -S docker docker-compose

# Add user to docker group
sudo usermod -aG docker $USER

# Enable and start Docker service
sudo systemctl enable docker
sudo systemctl start docker

# Install Podman as alternative
sudo pacman -S podman podman-compose

# Verify installation (after session restart)
docker --version
podman --version

terminal configuration

Configure an efficient terminal with customized shell and productivity tools.

zsh and oh-my-zsh

Configure Zsh as default shell with useful plugins:

# Install Zsh
sudo pacman -S zsh

# Install Oh My Zsh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

# Install useful plugins
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

# Configure plugins in ~/.zshrc
sed -i 's/plugins=(git)/plugins=(git zsh-autosuggestions zsh-syntax-highlighting)/' ~/.zshrc

command line tools

Install modern tools that improve the command line experience:

# Modern terminal tools
sudo pacman -S exa bat fzf tree htop neofetch

# Install starship prompt
yay -S starship

# Add starship to shell
echo 'eval "$(starship init zsh)"' >> ~/.zshrc

# Create useful aliases in ~/.zshrc
echo 'alias ll="exa -la"' >> ~/.zshrc
echo 'alias cat="bat"' >> ~/.zshrc
echo 'alias find="fd"' >> ~/.zshrc

databases

Configure databases for local development:

postgresql

Install and configure PostgreSQL for development:

# Install PostgreSQL
sudo pacman -S postgresql

# Initialize database
sudo -u postgres initdb -D /var/lib/postgres/data

# Enable and start service
sudo systemctl enable postgresql
sudo systemctl start postgresql

# Create user for development
sudo -u postgres createuser --interactive

# Create test database
sudo -u postgres createdb testdb

redis

Configure Redis for caching and queues:

# Install Redis
sudo pacman -S redis

# Enable and start service
sudo systemctl enable redis
sudo systemctl start redis

# Test connection
redis-cli ping

productivity tips

Configure some tools and workflows that significantly increase development productivity.

useful scripts

Create scripts to automate common tasks:

#!/bin/bash
# ~/scripts/dev-setup.sh

echo "Starting development environment..."

# Navigate to projects directory
cd ~/projects

# Activate Python virtual environment if it exists
if [ -d "venv" ]; then
    source venv/bin/activate
    echo "Python virtual environment activated"
fi

# Check if it's a Node.js project
if [ -f "package.json" ]; then
    echo "Node.js project detected"
    npm install
fi

# Check if it's a Rust project
if [ -f "Cargo.toml" ]; then
    echo "Rust project detected"
    cargo check
fi

echo "Development environment ready!"

tmux configuration

Configure tmux for organized development sessions:

# Install tmux
sudo pacman -S tmux

# Create basic ~/.tmux.conf configuration
cat > ~/.tmux.conf << 'EOF'
# Prefix key
set -g prefix C-a
unbind C-b
bind C-a send-prefix

# Split panes
bind | split-window -h
bind - split-window -v

# Reload config
bind r source-file ~/.tmux.conf \; display "Config reloaded!"

# Mouse support
set -g mouse on

# Vi mode
setw -g mode-keys vi

# Status bar
set -g status-bg black
set -g status-fg white
set -g status-left '#[fg=green]#S'
EOF

conclusion

This development environment provides a solid foundation for working with multiple languages and technologies on Arch Linux. The setup is modular, allowing you to add or remove components based on your specific needs.

Remember to keep your tools updated regularly with pacman -Syu and yay -Syu, and back up your important configurations, especially files in ~/.config/.

The next step would be to further customize the configurations based on the specific types of projects you develop most frequently.
